# 🚀 Full Stack Development Lab (FSDL)

## 📌 Project Overview
This repository contains all the assignments and mini-projects completed as part of the **Full Stack Development Lab (FSDL)** for **T.Y. B.Tech Computer Engineering**.

The lab focuses on building practical skills in:
- Frontend Development
- Backend Development
- Database Integration
- Full Stack (MERN) Applications 

---
 
## 🎯 Objectives
- Learn complete web development lifecycle  
- Build real-world applications  
- Understand client-server architecture  
- Work with databases (MongoDB)  
- Develop scalable full-stack applications  

---

- Static webpage showing all HTML text formatting tags
- Displayed in tabular format
