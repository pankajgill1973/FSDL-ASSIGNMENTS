# HTML Text Formatting Tags – Static Web Page

## 📌 Project Overview
This assignment is a static web page that demonstrates commonly used HTML text formatting tags in a tabular format. The table includes the tag name, its description, and an example of how the formatted text appears in a web browser.

The purpose of this project is to help understand how different HTML tags modify the appearance and meaning of text on a webpage.

---

## 🎯 Objective
The main objectives of this project are:

- To understand and use different HTML text formatting tags  
- To practice structuring information using HTML tables  
- To create a simple static webpage using HTML  

---

## 🏷️ HTML Formatting Tags Included

The webpage demonstrates the following text formatting tags:

- `<b>` – Bold text  
- `<strong>` – Important text  
- `<i>` – Italic text  
- `<em>` – Emphasized text  
- `<mark>` – Highlighted text  
- `<small>` – Smaller text  
- `<del>` – Deleted text  
- `<ins>` – Inserted text  
- `<sub>` – Subscript text  
- `<sup>` – Superscript text  

Each tag is displayed in the table along with its description and an example output.

---

## ▶️ How to Run the Project

1. Download or clone the project files  
2. Open the project folder  
3. Open the `index.html` file in any web browser  
4. The webpage will display a table explaining all HTML text formatting tags  

---

## 📚 Conclusion

This project helps beginners understand how HTML text formatting tags work and how they can be used to improve the readability and presentation of content on a webpage.
